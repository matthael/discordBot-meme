# 🤖 Discord Meme Bot

Um bot simples para Discord focado em enviar memes e interações divertidas no servidor.

## 🚀 Funcionalidades

- Envio automático de memes
- Comandos simples
- Respostas interativas
- Fácil de configurar

## 🛠️ Tecnologias Utilizadas

- Node.js
- Discord.js

## 📦 Instalação

1. Clone o repositório:

```bash
git clone https://github.com/matthael/discordBot-meme.git

Entre na pasta do projeto:

cd discordBot-meme

Instale as dependências:

npm install

Configure o arquivo .env com seu token do bot:

TOKEN=seu_token_aqui

Inicie o bot:

node index.js
📌 Observações

Certifique-se de ter o Node.js instalado.

Nunca compartilhe o token do seu bot.
